## Key Components

### 1. Notifier (The Ear)
- **File:** `src/workflow_orchestration_queue/notifier/webhook.py`
- **Framework:** FastAPI
- **Endpoint:** `POST /webhooks/github`
- **Security:** HMAC SHA256 signature verification
- **Purpose:** Ingests GitHub webhooks and creates WorkItems

### 2. Queue (The State)
- **Interface:** `src/workflow_orchestration_queue/interfaces/task_queue.py`
- **Implementation:** `src/workflow_orchestration_queue/queue/github_queue.py`
- **Backend:** GitHub Issues + Labels
- **State Machine:** queued → in-progress → success/error/infra-failure

### 3. Sentinel (The Brain)
- **File:** `src/workflow_orchestration_queue/sentinel/poller.py`
- **Pattern:** Async polling with jittered exponential backoff
- **Interval:** 60 seconds (configurable)
- **Concurrency:** Assign-then-verify distributed locking

### 4. Worker (The Hands)
- **Technology:** opencode-server CLI
- **Interface:** Shell bridge script (`scripts/devcontainer-opencode.sh`)
- **Workflows:** Markdown-based instruction modules
